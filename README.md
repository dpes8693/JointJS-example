## 其他

JointJS 是一个开源前端框架，支持绘制各种各样的流程图、工作流图等。Rappid 是 Joint 的商业版，提供了一些更强的插件
<https://www.cnblogs.com/helica/p/11354577.html>

    <!-- content end-->
